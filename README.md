# Sistema de Gestión de Inventario - SENA

Proyecto académico desarrollado para la **Ficha 3219160** del programa ADSO (Análisis y Software Development). Este sistema permite el control de productos mediante un entorno de consola basado en **Programación Orientada a Objetos (POO)**.

## 📋 Descripción del Proyecto
El software implementa un modelo de inventario capaz de registrar productos y gestionar su persistencia mediante la exportación de reportes en formato `.txt`. La arquitectura está diseñada para ser modular, facilitando el mantenimiento y la escalabilidad del código.

## 🚀 Características Principales
*   **Arquitectura Orientada a Objetos:** Uso de clases (`Producto` e `Inventario`) para una organización lógica de datos.
*   **Persistencia de Archivos:** Generación automatizada de reportes mediante el manejo de archivos (I/O).
*   **Interfaz de Usuario:** Menú interactivo sencillo y eficiente para la gestión de productos.
*   **Código Limpio:** Implementación de buenas prácticas y manejo de excepciones.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python 3.x
- **Paradigma:** POO (Programación Orientada a Objetos)
- **Entorno:** CLI (Command Line Interface)
